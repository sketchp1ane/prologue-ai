# 00 — Project Brief

## Product

**Prologue / 第一页** is an AI-powered job-search workspace for individual job seekers.

## User problem

Job seekers often manage resumes, job descriptions, applications, outreach, and interview notes in separate tools. They need one workspace that turns each application into a repeatable workflow.

## MVP scope

Build a 3-week shippable MVP with:

1. Authenticated workspace
2. Resume upload / paste-text fallback
3. Resume parsing into structured profile
4. Application board
5. JD extraction
6. Resume-vs-JD diagnosis report
7. Bullet rewrite with streaming variants
8. Outreach message generation
9. Interview review notes
10. Weekly application insight report
11. Deployment-ready README and demo data

## Out of scope for MVP

- Full resume editor
- Multi-user collaboration
- Payment
- Complex analytics dashboard
- Real-time notifications
- Chrome extension
- Automated job application submission
- Mock interview voice/video mode

## Success definition

A reviewer can open the deployed app, sign in, upload or paste a resume, create an application from a JD, generate a diagnosis report, rewrite a resume bullet, generate outreach copy, and see the application on a board.
